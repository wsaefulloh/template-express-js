## Installation Steps

1. Clone the repository

```bash
https://github.com/wsaefulloh/template-express-js.git
```

2. Install dependencies

```bash
npm install
```

3. Run the app

```bash
npm start
```

4. You are all set!

```bash
View the website at: http://localhost:9000
```
